单线程为什么还这么快？

  • 纯内存
  • 非阻塞IO
  • 避免线程切换和竞态消耗

其它注意点

  • 一次只运行一条命令
  • 拒绝长（慢）命令
  • 其实不是全是单线程，如下面的命令 fysnc file descriptor , close file descriptor
