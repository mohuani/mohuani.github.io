#### Java中的常见注解

JDK自带注解

- @Override，表示当前的方法定义将覆盖超类中的方法。

- @Deprecated，使用了注解为它的元素编译器将发出警告，过时被弃用的代码，不建议再使用。

- @SuppressWarnings，关闭不当编译器警告信息。

    例子：@SuppressWarnings("deprecation")，仍然使用已经不推荐的方法
