Mac上面关闭 双指左滑右滑，导致浏览器前进后退的现象

### Chrome

```
defaults write com.google.Chrome AppleEnableSwipeNavigateWithScrolls -bool false
```

### Microsoft Edge

```
defaults write com.microsoft.edgemac AppleEnableSwipeNavigateWithScrolls -bool false
```
